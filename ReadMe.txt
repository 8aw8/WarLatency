 КОНСОЛЬНОЕ ПРИЛОЖЕНИЕ. Обзор проекта WarLatency
  ========================================================================
Для компиляции и запуска приложения необходим 
1. Microsoft Visual Studio 2010 SP1 (Среда разработки, на которой разрабатывалось)
  
  
Сборка.
Открыть проект в Microsoft Visual Studio 2010 SP1 можно с помощью файла WarLatency.vcxproj.
Компиляция F6 или F7 (в зависимости от настройки), запуск F5.
  
Тестирование.
Запустить исходный проект в Microsoft Visual Studio 2010 SP1
В командном редакторе (cmd.exe) 
выполнить команду telnet 127.0.0.1 12345
  
Появится меню: 
 List gamers: 1
 Start game:  2
 Quit:        9
  
Для запуска игры выбрать цифру 2 и нажать ENTER.
